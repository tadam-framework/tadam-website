## Contribute

To contribute or **edit you must modify the markdown files** that you will find in the `/markdown/` folder.

## Compile

### Install

``` sh
npm install -g parcel-bundler
```

### Dev

``` sh
parcel index.pug --out-dir docs
```

### Production

``` sh
parcel build index.pug --out-dir docs
```
